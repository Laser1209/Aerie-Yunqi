# Aerie 灵动岛改造方案 v12.0.1

> 用桌面顶部的「灵动岛」替代原有悬浮球，打造沉浸式陪伴体验

---

## 一、项目背景与目标

### 1.1 背景

现有悬浮球方案存在以下问题：

- 视觉风格与产品定位（恋人陪伴）契合度不够
- 功能单一，缺乏扩展性
- 动画效果不够灵动治愈

### 1.2 目标

- 删除现有悬浮球全部代码
- 实现桌面顶部灵动岛（Dynamic Island）
- 组件化架构，用户可自由配置显示内容
- 多主题切换（深色 / 恋粉 / 浅白）
- 多种交互方式 + 多种展开形态
- 弹性果冻感动画 + 粒子特效

---

## 二、核心设计理念

### 2.1 视觉语言

- **核心关键词**：治愈、灵动、陪伴、呼吸感
- **设计原则**：极简形态 + 丰富动效 + 组件可配置
- **动画哲学**：每个状态变化都有对应的弹性过渡，像有生命一样

### 2.2 四态设计

| 状态               | 说明                             | 触发条件        |
| ------------------ | -------------------------------- | --------------- |
| **胶囊态**   | 默认收起的小药丸，显示核心信息   | 日常待机        |
| **通知态**   | 横向展开显示通知，几秒后自动收回 | 收到新消息/事件 |
| **工具栏态** | 悬停时展开，显示快捷功能按钮     | 鼠标悬停        |
| **展开态**   | 完整面板，显示详细内容和操作     | 点击/长按       |

## 五、交互方式（可配置）

### 5.1 触发展开方式

| 模式               | 说明                               | 适用场景               |
| ------------------ | ---------------------------------- | ---------------------- |
| **点击展开** | 单击胶囊 → 展开，再次点击 → 收起 | 经典灵动岛，误触少     |
| **悬停展开** | 鼠标移入立即展开，移出自动收回     | 快捷操作，效率高       |
| **两级交互** | 悬停显示工具栏，点击才展开完整面板 | 平衡效率与误触（推荐） |
| **长按展开** | 长按展开，跟手机一致               | 防误触最佳             |

### 5.2 其他手势

- **拖拽移动**：长按后可拖动到屏幕顶部任意位置（左/中/右）
- **右键菜单**：设置、隐藏、切换主题
- **滚轮调节**：滚轮切换胶囊态显示的组件

## 六、展开形态（可配置）

### 6.1 工具栏形态

- 展开宽度：~300px
- 内容：3-5个圆形图标按钮
- 适用：快捷操作

### 6.2 控制面板形态（推荐）

- 展开宽度：~400-500px
- 内容：标题 + 2-3行组件内容 + 操作按钮
- 适用：大多数场景

### 6.3 侧边栏形态

- 展开宽度：~320px，从顶部向下延伸
- 内容：完整的小部件面板
- 适用：功能重度用户

### 6.4 通知形态

- 仅通知时展开，3-5秒后自动收回
- 内容：图标 + 标题 + 简短内容
- 适用：消息提醒

## 七、动画系统

### 7.1 弹性果冻感动画

- 动画时长：500ms（展开）/ 400ms（收起）
- 缓动曲线：`cubic-bezier(0.34, 1.56, 0.64, 1)`（弹性回弹）
- 双重动画：尺寸变化 + 轻微缩放回弹（1.04 → 1.0）

### 7.2 动画时间轴（展开时）

```
0ms      ── 开始，胶囊轻微放大（1.04倍）
100ms    ── 开始横向扩展
300ms    ── 横向扩展完成，开始纵向扩展
400ms    ── 纵向扩展完成，内容淡入
500ms    ── 缩放回弹结束，动画完成
```

### 7.3 粒子特效

| 场景       | 特效                                |
| ---------- | ----------------------------------- |
| 新消息通知 | 粉色粒子从胶囊中散出，像撒糖一样    |
| 心情变化   | 对应颜色的光晕呼吸效果              |
| 点击展开   | 轻微的光点爆散效果                  |
| 陪伴状态   | 缓慢呼吸的光晕 + 偶尔飘出小爱心粒子 |

### 7.4 内容过渡动画

- 组件切换：淡入淡出 + 轻微位移（±4px）
- 文字滚动：平滑向左滚动（跑马灯效果）
- 图标变化：缩放旋转过渡

## 十一、风险与注意事项

| 风险                 | 影响         | 解决方案                                         |
| -------------------- | ------------ | ------------------------------------------------ |
| Windows 透明窗口残影 | 视觉体验差   | 关闭硬件加速 + 启用透明视觉 + DWM 合成调优       |
| 高频鼠标穿透切换卡顿 | 系统卡顿     | 主进程防抖（80ms）+ 批量操作                     |
| 动画性能不佳         | 掉帧         | 使用 transform + opacity 动画，避免 layout/paint |
| 多显示器适配         | 位置错乱     | 检测光标所在显示器，灵动岛跟随                   |
| 全屏应用遮挡         | 看不到灵动岛 | `visibleOnAllWorkspaces + floating` 层级       |
