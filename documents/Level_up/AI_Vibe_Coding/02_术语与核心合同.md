# 术语与核心合同
> [!abstract] 核心
> Conversation、Turn、Message、Request 是聊天真源；World 仅经 WorldPort 返回版本化 DTO。

|术语|合同|
|---|---|
|Actor|跨 Channel 的长期记忆主体|
|Channel|desktop 或 QQ；短期 Conversation 隔离边界|
|Turn|完整的一轮用户输入与助手输出|
|Message|可展示气泡；同组由 response_group_id 与 sequence 排序|
|Request|可取消、重试并持久化状态的处理请求|
|EventEnvelope|统一携带 event/request/conversation/turn/message/group/sequence/channel|
|ImageCandidate|World 的建议 DTO，不执行图片副作用|
